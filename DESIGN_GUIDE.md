# 펫비타 디자인 가이드

> 참조 사이트: [pethroom.com](https://pethroom.com) — 프리미엄 반려동물 케어 브랜드의 세련되고 신뢰감 있는 라이프스타일 쇼핑 플랫폼
> 목표: 따뜻함 + 현대성의 균형, 구독 서비스에 최적화된 UX, 정제된 시각 언어

---

## 1. 색상 시스템 (Color Palette)

### 1-1. 브랜드 컬러
| 용도 | HEX | 설명 |
|------|-----|------|
| **Primary (Deep Navy)** | `#003366` | 제목, 강조 텍스트, 브랜드 아이덴티티 |
| **Accent (Warm Orange)** | `#FF9900` | CTA 버튼, 할인가, VIP 강조 |
| **White** | `#FFFFFF` | 기본 배경, 상품 썸네일 배경 |

### 1-2. 중립 컬러
| 용도 | HEX |
|------|-----|
| Background Gray | `#F5F5F5` |
| Body Text | `#333333` |
| Muted Text | `#888888` |
| Border | `#E5E5E5` |
| Sub-highlight (연 파랑) | `#E8F4F8` |

### 1-3. 사용 원칙
- **주황 × 네이비**를 두 축으로 시각적 리듬 생성
- 할인가는 주황, 정상가는 회색 취소선, VIP가는 굵은 주황으로 **3단계 가격 계층**
- 섹션 배경은 기본 화이트 / 콘트라스트용 `#F5F5F5` 교차 사용

## 2. 타이포그래피 (Typography)

### 2-1. 폰트 패밀리
```css
font-family: 'Noto Sans KR', 'Pretendard', 'Apple SD Gothic Neo', sans-serif;
```
한글 가독성 최우선. 영어 제목은 동일 폰트의 Light/Regular로 표현.

### 2-2. 크기 & 굵기 스케일
| 계층 | 크기 | 굵기 | 행간 |
|------|------|------|------|
| Display (Hero H1) | 40–48px | 900 | 1.25 |
| H2 섹션 제목 | 24–28px | 700 | 1.3 |
| H3 카드 제목 | 16–18px | 700 | 1.4 |
| Body | 14–16px | 400 | 1.6–1.8 |
| Caption / 소형 | 12px | 400 | 1.5 |

### 2-3. 사용 원칙
- **자간은 기본(normal)**, 영문 대문자 섹션 라벨에만 `letter-spacing: 0.08em`
- 섹션 제목은 **왼쪽 정렬이 기본**, 브랜드 메시지·CSR 섹션만 가운데
- Display 텍스트는 **두 줄 구성**으로 호흡을 주고 줄바꿈 지정

## 3. 레이아웃 구조 (Layout)

### 3-1. 페이지 흐름
```
[Header / Fixed]
  └ 로고 · 1차 메뉴 · 유틸 아이콘 · 장바구니

[Top Banner Slider]            ← 이벤트/프로모션
[Hero / Brand Story]           ← 좌측 정렬 텍스트 + 어워드 로고
[VIP / Subscription Benefits]  ← 혜택 카드 그리드
[Best Seller]                  ← 상품 슬라이더 (Swiper)
[New Arrival]                  ← 상품 슬라이더
[Curation — 영양제 라인]       ← 2x4 상품 그리드
[Review Slider]                ← 별점 + 회차별 후기
[IoT / Smart Care]             ← 제품 이미지 + 차트
[CSR — 유기동물 프로젝트]      ← 이미지 그리드
[CTA Banner]                   ← 풀블리드 주황/네이비 배너
[Footer / Dark]                ← 회사정보 · 정책 · 고객센터
```

### 3-2. 그리드 & 컨테이너
- **최대 너비**: 1200–1280px, `margin: 0 auto`
- **내부 여백**: `padding: 0 24px` (모바일 16px)
- **섹션 상하 여백**: `80–120px` (섹션 간 호흡 여유)
- **카드 간 gap**: `20–30px`
- **제목과 본문 사이**: `15–20px`

### 3-3. 반응형 브레이크포인트
| 디바이스 | 폭 | 상품 카드 열 |
|----------|----|----|
| Mobile | ~767px | 1–2 |
| Tablet | 768–1023px | 2–3 |
| Desktop | 1024px+ | 4 |

## 4. 섹션별 콘텐츠 패턴

### 4-1. 히어로 (Brand Story)
- 좌측 정렬 긴 메시지 + 어워드 로고 4개(Red Dot, German Design Award 등)
- 미니멀한 텍스트 중심, 과한 이미지 없음
- 서브 카피는 철학 중심 ("반려동물의 입장에서 생각하고…")

### 4-2. VIP / 멤버십 섹션
- 8개 혜택을 **아이콘 + 짧은 설명** 카드 그리드로 표현
- 대표 혜택(35% 할인, 5% 적립 등)은 숫자를 크게 강조
- 하단 중앙 CTA: "정기배송 시작하기"

### 4-3. 상품 슬라이더
- Swiper.js 기반, **자동 재생 5s / 무한 루프**
- 화살표 + 도트 페이지네이션
- 카드 하나당 이미지 + 이름 + 설명 + 3단계 가격

### 4-4. 후기 슬라이더
- 별점 5개(★★★★★) 주황색
- 닉네임은 일부 마스킹 (예: "김\*\*")
- 회차 정보 표기 ("정기배송 3회차")

### 4-5. CSR / 브랜드 철학
- 가운데 정렬, 사진 그리드 6개
- 따뜻한 톤 실사진(반려동물 중심)

## 5. 컴포넌트 패턴 (Component Library)

### 5-1. 버튼
```css
/* Primary CTA */
background: #FF9900;
color: #FFFFFF;
border-radius: 4–6px;   /* 샤프한 둥근 사각 */
padding: 14px 28px;
font-weight: 700;
height: 48–56px;

/* Secondary */
background: transparent;
border: 1px solid #003366;
color: #003366;
```
> **Note**: pethroom은 pill(완전 원형)보다 **살짝 둥근 사각 (4–6px)** 을 선호. 모던하면서 단단한 인상.

### 5-2. 상품 카드
```
┌─────────────────┐
│  [이미지]       │  ← 1:1, border-radius 8–12px
│  (흰 배경)      │
├─────────────────┤
│ [BEST] [NEW]    │  ← 선택적 배지
│ 상품명 (16px)   │
│ 설명 (12px)     │
│ ₩29,000 (취소)  │  ← #888 회색
│ ₩24,000 (검정)  │  ← 일반 할인가
│ VIP ₩19,000     │  ← 주황 강조
└─────────────────┘
```

### 5-3. 배지 (Badge)
- `NEW`: 파란(#003366) 배경 + 흰 텍스트, 작은 원형/알약
- `BEST`: 주황(#FF9900) 배경 + 흰 텍스트
- `VIP`: 주황 + 왕관/라벨 아이콘
- `품절`: 이미지 위 `rgba(0,0,0,.4)` 오버레이 + 중앙 텍스트

### 5-4. 네비게이션
- **상단 헤더**: 고정 / 스크롤 시 그림자 / 투명→화이트 전환
- **드롭다운**: 마우스오버 시 하위 카테고리 펼침 (대형 메가메뉴 아님, 간결한 리스트)
- 유틸 아이콘: 정기배송 · 프렌즈 · 검색 · 장바구니

### 5-5. 슬라이더 (Swiper)
- 도트 인디케이터 (간결/미니멀)
- 좌우 화살표 원형 버튼
- Autoplay 5초, 무한루프

### 5-6. 폼 요소
- 입력 필드: `border: 1px solid #E5E5E5`, radius 4px, height 48px
- 포커스 시 `border-color: #003366`

## 6. 비주얼 / 이미지 스타일

- **상품 사진**: 흰색 배경, 정중앙 정렬, 1:1 크롭
- **브랜드/CSR 사진**: 따뜻한 톤, 자연광 실사진, 반려동물 중심 클로즈업
- **일러스트**: 최소화 — 아이콘(SVG)만 사용 (선형 스타일, 얇은 스트로크)
- **여백**: 과하다 싶을 만큼의 화이트 스페이스가 프리미엄 감성 핵심
- **배경 장식**: 과한 패턴 없음. 필요시 옅은 그라디언트 블러 원형 하나 정도

---

## 7. 톤앤매너 (Voice & Tone)

### 7-1. 문구 스타일
- **정중하고 부드러움**: "~해보세요", "~겠습니다"
- **철학 중심 표현**: "반려동물의 입장에서", "행복해지는 그 날까지"
- **신뢰 강조**: 할인가 3단계 명시, 어워드·인증 노출

### 7-2. 브랜드 인상
- **프리미엄 캐주얼**: 고급스러우면서 친근
- **따뜻함 + 기술감**: CSR의 감성 + IoT의 모던함 공존
- **투명성**: 가격/성분/정책을 숨기지 않고 노출

### 7-3. 이모지 사용
- UI 상 이모지 최소화, 기능 아이콘은 **선형 SVG** 사용
- 이벤트 배너에서만 제한적 허용

---

## 8. 인터랙션 & 모션

- 카드 hover: `translateY(-4px) + box-shadow 강화`
- 버튼 hover: 색상 어둡게(-10%) + 약간의 상승
- 슬라이더: `ease-in-out`, 600ms
- 스크롤 진입: `fade + translateY(32px)` 온/오프

## 9. 펫비타 적용 가이드

현재 `index.html`/`styles.css`는 **그린 중심(#4F7942) + 앰버 포인트(#F5A623)** 의 자연친화적 팔레트.
pethroom 스타일로 리브랜딩하려면 다음을 조정:

| 항목 | 현재 | pethroom 스타일 |
|------|------|------------------|
| Primary | `#4F7942` (Forest Green) | `#003366` (Deep Navy) |
| Accent | `#F5A623` (Amber) | `#FF9900` (Warm Orange) |
| 버튼 radius | `50px` (pill) | `4–6px` (둥근 사각) |
| 카드 radius | `16px` | `8–12px` |
| 섹션 제목 정렬 | 가운데 | **왼쪽 정렬** |
| 가격 표기 | 단일 | **3단계 (정상/할인/VIP)** |
| 히어로 비주얼 | 떠다니는 카드 | 어워드 로고 + 텍스트 중심 |
| 상품 리스트 | 플랜 3카드 | **상품 슬라이더(Swiper) + 그리드** |
